# 反AI检测文风指南

本指南基于AI优化会议报告，提供具体的反AI检测写作技巧和检查清单。

---

## 核心原则

**AI检测器寻找的规律化特征，必须有意识打破**

AI写作的典型特征：
- 句式过于规整，长度均匀
- 高频使用"此外/然而/综上所述"等连接词
- 笼统描述多于具体数字
- 缺少情绪标点化
- 配角工具化，无个性
- 画面感薄弱

本指南提供12项具体技巧来打破这些特征。

---

## 技巧1：情绪标点化（强制）

### 感叹号制造节奏

| AI写法 | 人工写法 |
|--------|----------|
| [此房是我买，此路是我开。萧也，我来了。] | [此房是我买此路是我开...萧也我！来！了！] |
| 他很高兴。 | 他！！！高！！！兴！！！了！！！ |
| 太危险了。 | 太危险了！！！！（四个感叹号是灵魂） |

### 方括号内心弹幕格式

```
[少女请你记住！美丽的外表下往往隐藏着极致的危险！]
[大意了。跑。]
[此房是我买此路是我开...萧也我！来！了！]
[空中蹬车表演艺术家]
```

### 省略号制造停顿

```
"萧也...我来了..."  （停顿感）
不是："萧也，我来了。" （平淡）

"大意了...跑..."  （紧张喘息感）
不是："大意了，快跑。" （太流畅）
```

### 情绪标点使用频率

- **紧张场景**：每2-3句使用1次感叹号或省略号
- **喜剧场景**：大量使用感叹号、重复标点
- **温情场景**：省略号、波浪号（~）增加柔软感

## 技巧2：具体数字代替笼统描述（强制）

### 数字替换表

| 笼统描述（AI常见） | 具体数字（人工风格） | 使用场景 |
|-------------------|---------------------|----------|
| 他很累 | 48小时没合眼了，黑眼圈像熊猫 | 主角状态 |
| 他很高 | 一米八七，比门框还高半个头 | 外貌描写 |
| 人很多 | 黑压压三百多号人 | 场面描写 |
| 等了很久 | 抽完了三根烟，手机屏幕亮了又暗 | 时间流逝 |
| 很帅 | 9分男（内心OS打分） | 外貌评价 |
| 猫很胖 | 扁得像块被擀过的饺子皮 | 动物描写 |
| 耳朵贴平 | 变成了两架小小的飞机 | 动物动作 |
| 毛发蓬松 | 像刚出炉的蜂蜜面包 | 触感描写 |

### 执行要求

**每个状态描写必须包含具体数字或量化描述**

检查方法：用荧光笔标出所有形容词，替换成数字。

❌ 原文："他很累，脸色很差。"
✅ 修改："48小时没合眼了，脸色惨白得像张纸。"

## 技巧3：旁观者视角烘托（强制）

### 内心OS打分法

```
前台医生内心尖叫：7分→9分男！！！！
不是：萧也很帅

护士小声嘀咕：这怕不是个傻子吧？
不是：主角行为很奇怪

路人甲：这人怎么在空中蹬车？
不是：主角的动作很滑稽
```

### 配角即时反应

```
大橘："猫不理解，但猫尊重。"
前台："（眼睛发亮）需要帮忙吗？"
室友："我操......"（遥控器掉在地上）
```

### 多视角切换

```
【主角视角】
萧也走进医院，浑身湿透。

【医生视角】
前台医生抬头，瞳孔地震：这、这是9分男？！！

【猫视角】
大橘：两脚兽怎么变成落汤鸡了？

【上帝视角】
雪夜，医院门口，一人一猫，像电影画面。
```

## 技巧4：极简短句塑造性格（配角三句话法则）

### 三句话法则

**每个有台词的配角必须设计3句标志性短句**

| 角色 | 标志性短句1 | 标志性短句2 | 标志性短句3 | 性格标签 |
|------|------------|------------|------------|----------|
| 大橘 | 大意了。跑。 | 让人喂你，明天咪找你。 | 猫不理解，但猫尊重。 | 傲娇、机智 |
| 系统 | 【叮！检测到...】 | 【宿主，你清醒一点】 | 【奖励已到账，请查收】 | 毒舌、高效 |
| 前台医生 | 7分→9分男！！！！ | 这位先生，请填写表格。 | （眼睛发亮）需要帮忙吗？ | 花痴、专业 |
| 室友A | 脸能当球看吗？ | 不是吧，你来真的？ | 服了服了。 | 嘴贱、怂快 |

### 对比示例

| AI写法（冗长） | 人工写法（极简） |
|---------------|-----------------|
| 大橘发现自己大意了，决定快点跑掉。 | 大意了。跑。 |
| 猫虽然不理解人类的行为，但还是选择尊重他。 | 猫不理解，但猫尊重。 |
| 医生觉得这个男人非常帅，忍不住多看了几眼。 | 9分男！！！！ |

### 短句使用场景

- **逃跑/紧张时**：极简短句（2-5字）
- **傲娇/吐槽时**：方括号+短句
- **震惊/感叹时**：短句+多感叹号

## 技巧5：组合信号写动作（强制）

### 动作分解公式

```
[整体动作] = [部位1动作] + [部位2反应] + [声音/效果]

示例：白猫跳上窗台
= 后腿蹬地（起跳）+ 前爪扒住窗沿（抓握）+ 尾巴僵直（平衡）+ 发出"咪呜"一声（声音）
```

### 组合信号示例

**猫的动作**：
```
❌ AI：猫很害怕地挣扎
✅ 人工：蹬腿（挣扎）+ 飞机耳（害怕）+ 炸毛（防御）+ 发出嘶嘶声（警告）

描写：
白猫后腿乱蹬，耳朵"唰"地贴平，变成了两架小小的飞机。背上的毛根根竖起，发出"嘶——"的低吼。
```

**人的动作**：
```
❌ AI：他很紧张
✅ 人工：手指敲桌（焦虑）+ 喉咙吞咽（口干）+ 眼神闪烁（心虚）+ 深呼吸（平复）

描写：
他手指在桌面上敲击，越来越快。喉咙发干，咽了口唾沫。眼神飘忽不定，最后深吸一口气，像是下定了什么决心。
```

### 常用动作信号对应表

| 情绪 | 信号组合 |
|------|----------|
| 紧张 | 手指敲桌 + 喉咙吞咽 + 眼神闪烁 |
| 害怕 | 后退 + 抱臂 + 眼神躲避 |
| 愤怒 | 握拳 + 咬牙 + 呼吸加重 |
| 害羞 | 低头 + 耳朵红 + 绞手指 |
| 震惊 | 瞳孔放大 + 张嘴 + 身体僵硬 |

## 技巧6：生活化比喻（画面感核心）

### 喻体词库

**食物类**：
- 饺子皮（形容扁平）
- 蜂蜜面包（形容蓬松/金黄）
- 布丁（形容Q弹/柔软）
- 棉花糖（形容柔软/白色）
- 糯米团子（形容圆润/可爱）
- 芝麻球（形容黑色/圆润）

**日常物品类**：
- 飞机（形容耳朵贴平）
- 遥控器（形容手指姿势）
- 手机（形容发光/方形）
- 拖鞋（形容形状/随意）
- 闹钟（形容吵闹）

**自然类**：
- 蒲公英（形容毛发蓬松）
- 煤球（形容黑色/圆形）
- 熊猫（形容黑眼圈）
- 企鹅（形容走路姿势）
- 树懒（形容慢吞吞）

### 使用要求

**每章至少使用2-3个生活化比喻**

检查清单：
- [ ] 有食物类比喻吗？
- [ ] 有日常物品类比喻吗？
- [ ] 有自然类比喻吗？

## 技巧7：电影镜头式描写（强制）

### 每段扩写前自问

**这段如果拍成电影，画面是什么？**

### 镜头语言要素

```
【特写】→ 表情、手部动作、物品细节
【中景】→ 人物互动、肢体语言
【远景】→ 环境、氛围、剪影
【逆光】→ 营造氛围、神秘感
【侧光】→ 突出轮廓
```

### 镜头切换示例

```
【特写】萧也的手指悬在门把手上，指节发白。

【中景】他站在医院门口，浑身湿透，水珠顺着下巴滴落。

【远景】雪夜，高大的男人逆着光，坚定地朝里面走来。

【特写】白猫耳朵"唰"地贴平，变成了两架小小的飞机。

【逆光】地上的白猫扁得像块被擀过的饺子皮，逆着光看像个毛茸茸的影子。
```

### 与比喻结合

```
【远景+比喻】
雪夜里，萧也的身影像从电影里剪出来的，大衣下摆翻飞如乌鸦的翅膀。

【特写+比喻】
白猫扁得像块被擀过的饺子皮，只有两只眼睛还亮晶晶的。
```

## 技巧8：句式多样性

### 长短句混合

```
短句（5-10字）：制造节奏感
长句（30-40字）：展开细节

示例：
他笑了。（短句）
那是一种复杂的笑，带着三分嘲讽，三分不屑，还有四分说不清道不明的意味。（长句）
没有理由。（短句）
就是能赢。（短句）
```

### 语态转换

```
被动改主动：
- 数据被分析 → 我们分析了数据

主动改被动（偶尔）：
- 他说 → 话从他嘴里说出来
- 猫跑了 → 一道橘色闪电窜了出去
```

### 句式错位

```
倒装句：
从窗外望去，城市灯火阑珊。

插入语：
这球——他眯起眼睛——必进无疑。

句子碎片：
明白了。就这么干。没有退路了。

语序颠倒：
走了啊我。（代替：我走了啊）
吃饭去咱们。（代替：咱们吃饭去）
```

## 技巧9：避免AI高频连接词

### 替换表

| AI高频词 | 人工替换方案 | 使用场景 |
|---------|------------|---------|
| 此外/另外 | 再说、还有一点、话说回来 | 补充信息 |
| 然而/因此 | 但、所以、不过呢、问题是 | 转折/因果 |
| 综上所述 | 说白了、老实说、说到底 | 总结 |
| 非常重要 | 很关键、这事不能忽视、说白了 | 强调 |
| 与此同时 | 这边...那边...、一边...一边... | 并行 |
| 值得注意的是 | 有意思的是、别说、你看 | 引出重点 |
| 换句话说 | 说白了就是、意思就是说 | 解释 |
| 从某种程度上说 | 说实话、讲真、老实说 | 缓和语气 |
| 毫无疑问 | 那当然、还用说、必须的 | 肯定 |
| 总的来说 | 一句话、说白了、总结下 | 总结 |

### 检查方法

用搜索功能检查以下词汇，全部替换：
- 此外、另外、而且
- 然而、但是（可保留"但"）
- 因此、所以（可保留"所以"）
- 综上所述
- 值得注意的是
- 换句话说
- 从某种程度上说

## 技巧10：吐槽密度控制

### 占比要求

| 场景类型 | 吐槽句数/段 | 占比 |
|----------|------------|------|
| 紧张场景 | 1-2句 | 10% |
| 过渡场景 | 3-4句 | 20% |
| 喜剧场景 | 5-8句 | 40-50% |
| 整体章节 | - | 20-30% |

### 吐槽格式

**方括号内心弹幕**：
```
[少女请你记住！美丽的外表下往往隐藏着极致的危险！]
[空中蹬车表演艺术家]
[此房是我买此路是我开...萧也我！来！了！]
```

**内心OS**：
```
他心想：这怕不是个傻子吧？
医生内心尖叫：9分男！！！！
```

**自我吐槽**：
```
我这是在干什么...
完了，这下全完了。
这是什么人间疾苦。
```

## 技巧11："意外+反应"喜剧公式

### 节奏模板

```
紧张 → 意外 → 喜剧 → 温情
```

### 示例

```
【紧张】
猫被尾巴绊倒，眼看要摔下窗台。

【意外】
慌乱中一爪子拍掉了萧也的帽子。

【喜剧】
前台医生看到"9分男"尖叫：卧槽！！！

【温情】
雪夜里，萧也抱着猫，大衣裹住两个小东西。
```

### 每章至少1处

检查清单：
- [ ] 有紧张铺垫吗？
- [ ] 有意外的转折吗？
- [ ] 有喜剧释放吗？
- [ ] 有温情收尾吗？

## 技巧12：增加"人味"特征

### 口语化表达

**语气词使用**：
```
"哦"、"呢"、"啊"、"嘛"、"呗"、"啦"、"咯"

示例：
"这能行吗？"他嘀咕着，"我总觉得不太靠谱呢。"
"就这样吧，随便啦。"
"你来啊，怕你啊？"
```

### 轻微瑕疵（故意）

**轻微重复**：
```
他想了想，又想了想，还是决定...
等等，等等，让我理一下。
```

**口语化啰嗦**：
```
这块可能说乱了，意思就是...
怎么说呢，就是那个...
```

**小幅语法跳跃**：
```
短句不完整："明白了。就这么干。"
语序颠倒："走了啊我。" "吃饭去咱们。"
```

### 主观表达

```
使用第一人称感受：
- "我觉得..."
- "说实话..."
- "老实说..."
- "讲真..."

加入个人判断：
- "这招挺损的，但好用。"
- "说实话，我觉得不行。"
- "讲真，这操作我服了。"
```

### 反AI检测检查（关键！必须全部通过）

- [ ] **情绪标点化**：激动处使用感叹号，可重复（！！！）
- [ ] **具体数字**：每个状态描写都有数字（48小时、9分男、三百多号人）
- [ ] **旁观者视角**：至少1处配角内心OS打分或吐槽
- [ ] **极简短句**：配角有3句标志性短句（如"大意了。跑。"）
- [ ] **组合信号**：动作描写包含≥3个维度（部位+反应+声音）
- [ ] **生活化比喻**：至少2处生活化比喻（饺子皮、飞机耳、蜂蜜面包）
- [ ] **电影画面感**：每段都能想象出电影镜头
- [ ] **无AI高频连接词**（此外/然而/综上所述/值得注意的是/从某种程度上说）

- [ ] **吐槽密度**：喜剧场景5-8句/段，紧张场景1-2句/段
- [ ] **意外+反应**：至少1处"意外+反应"喜剧节奏

## 扩写示例（反AI检测版）

### 示例F：喜剧场景（综合应用）

**骨架**：
```
主角带猫去医院，遇到前台医生 → [对话][动作][心理][吐槽]
```

**扩写示范**：
```
【特写】
萧也的手指还没碰到门把手，怀里的白猫突然炸了毛。

"咪呜——！！！"

【中景】
那声音惨得像被踩了尾巴——虽然它确实被自己的尾巴绊了一下。

白猫后腿乱蹬，前爪在空中抓挠，活像个空中蹬车表演艺术家。

[大意了。跑。]

萧也脑子里突然闪过这三个字，也不知道是猫的想法还是他自己的。

【意外】
慌乱中，猫爪子精准地拍掉了萧也的帽子。

【远景】
雪夜里，医院大厅的灯光惨白。一个浑身湿透的高大男人，顶着一头乱发，怀里抱着一只正在"蹬车"的白猫。

【特写+旁观者视角】
前台医生抬起头，目光在触及萧也脸的瞬间——

瞳孔地震。

内心尖叫：7分→9分男！！！！

【喜剧】
"这位...先生？"医生的声音都在抖，"您、您需要帮忙吗？"

[需要！非常需要！需要你的微信号！]

当然，这话她没说出口。

【温情】
萧也低头看了眼怀里的猫。白猫已经不蹬了，扁得像块被擀过的饺子皮，只有两只眼睛还亮晶晶的。

"嗯。"他说，"看病。"

【特写】
医生接过猫的时候，手指"不经意"地擦过萧也的手背。

[猫不理解，但猫尊重。]

白猫把耳朵贴平，变成了两架小小的飞机。
```

**技巧清点**：
- ✅ 情绪标点化（！！！、——）
- ✅ 具体数字（7分→9分）
- ✅ 旁观者视角（医生内心OS）
- ✅ 极简短句（大意了。跑。）
- ✅ 组合信号（蹬腿+飞机耳+炸毛）
- ✅ 生活化比喻（饺子皮、飞机）
- ✅ 电影镜头式（特写→中景→远景）
- ✅ 无AI连接词
