### 每章要求
- ✅ 字数：**3000+ 字**（硬性要求）
- ✅ 段落：12+ 段
- ✅ 场景：3+ 个
- ✅ 钩子：章尾必须有悬念

### 文风参考（写作前必读）

2. **`00-元数据/反AI检测文风指南.md`** - AI检测规避技巧（关键！）

1. **必须先阅读 `反AI检测文风指南.md`**，确保输出通过AI检测

4. **使用 `/novel meeting`** 做重要决策
5. **及时更新数据**（Lorekeeper会自动处理）
