## 扩写示例参考

根据场景类型选择对应的扩写策略：

### 示例A：都市对话场景
**适用场景**：现代都市、职场、社交场合

骨架：主角遇到老同学
```
主角在咖啡厅遇到多年未见的老同学 → [对话][心理][动作]
```

扩写要点：
- **[对话]** 寒暄中带试探，话里有话
- **[心理]** 回忆涌上心头，对比今昔
- **[动作]** 细微表情变化（眼神闪躲、手指摩挲杯沿）
- **[节奏]** 先慢后快，情绪递进

扩写示范：
```
"好久不见。"

声音从背后传来时，主角的手指正悬在咖啡杯沿上。那声音带着熟悉的慵懒，像是从大学宿舍的某个午后直接穿越而来。

他转过身，看见一张既熟悉又陌生的脸。

林默。

这个名字在舌尖转了一圈，终究没有叫出口。五年了，足够让一个人从青涩变成世故，从张扬变得收敛。眼前的林默穿着剪裁得体的西装，领带打得一丝不苟，和记忆里那个总是T恤配短裤的少年判若两人。

"是啊，好久不见。"主角笑了笑，指了指对面的座位，"坐？"

"不了，赶时间。"林默摆摆手，目光在主角身上停留了不到一秒，"听说你现在混得不错？"

这话说得随意，却像是在探什么底。主角注意到他的手指正无意识地转动着腕上的手表——那是他们大学时约定过的"紧张小动作"。

原来他也紧张。

"还行吧，混口饭吃。"主角轻描淡写地带过，"你呢？"

"也就那样。"林默笑了笑，那笑容没有到达眼底，"改天聚？"

"好，改天聚。"

两人都知道这句"改天"意味着什么。主角看着林默转身离去的背影，忽然想起大四那年，他们躺在宿舍楼顶看星星，林默说要做一辈子的兄弟。

咖啡已经凉了。
```

### 示例B：动物视角场景
**适用场景**：萌宠文、奇幻文、特殊视角叙事

骨架：小猫观察人类
```
小猫趴在窗台上观察楼下的人类 → [心理][感官][动作]
```

扩写要点：
- **[心理]** 使用方括号标记内心吐槽：[这个两脚兽在做什么？]
- **[感官]** 放大嗅觉、听觉（脚步声、气味变化）
- **[动作]** 猫科动物特有的动作（耳朵转动、尾巴轻摆、瞳孔收缩）
- **[视角]** 从低处看世界，物品显得高大

扩写示范：
```
阳光正好。

小白猫把下巴搁在前爪上，眯着眼睛看窗外。楼下那个穿红衣服的两脚兽又在原地转圈了，手里还拿着一个会发光的板子。

[又在拍什么？]

它歪了歪头，耳朵随着楼下传来的声音轻轻转动。那两脚兽的笑声太刺耳了，像是爪子挠在玻璃上的声音。

小白猫打了个哈欠，露出粉嫩的舌头。

[无聊。]

它决定换个姿势。后腿用力一蹬，整只猫就转了个方向，毛茸茸的尾巴在空中划出一道优美的弧线。这个动作它练了很久，现在可以做得行云流水。

楼下突然传来一声尖叫。

小白猫的耳朵"唰"地竖了起来，瞳孔瞬间收缩成一条细线。它猛地站起来，前爪扒住窗框，鼻子几乎要贴到玻璃上。

[出事了？]

原来是那个两脚兽踩到了水坑。小白猫看着她在原地蹦跳的样子， slowly地把耳朵转回了放松的状态。

[笨蛋。]

它重新趴下来，把尾巴卷成一个问号形状。阳光晒得背上的毛暖烘烘的，让人——让猫昏昏欲睡。

楼下又传来脚步声，这次是从楼道里传来的。小白猫的鼻子动了动，捕捉到了熟悉的气味。

[他回来了。]

它跳下窗台，爪子落地时没有发出一点声音。然后端端正正地坐在门边，尾巴整齐地缠住前爪，等待。

等待那个会给它开罐头的人。
```

### 示例C：情感冲突场景
**适用场景**：感情戏、争吵、内心挣扎

骨架：两人发生争执
```
主角和好友因为误会争吵 → [对话][动作][心理]
```

扩写要点：
- **[对话]** 潜台词，不说真话；打断、转移话题、答非所问
- **[动作]** 肢体语言代替直接表达（握紧拳头、避开眼神）
- **[节奏]** 短句制造紧张感，长句放缓情绪
- **[心理]** 内心真实想法与表面态度的反差

扩写示范：
```
"你到底什么意思？"

声音压得很低，像是从牙缝里挤出来的。主角盯着对面的苏晴，手指无意识地攥紧了桌沿。

苏晴没说话，只是把脸转向了窗外。

沉默。

咖啡厅里的音乐还在响，是一首轻快的爵士乐，和此刻的气氛格格不入。主角觉得那旋律刺耳得很。

"说话。"

"我说什么？"苏晴终于转过头，眼眶是红的，"你都认定了，我还说什么？"

"我认定什么了我？"

"你认定是我说的。"苏晴笑了，那笑容比哭还难看，"你刚才那眼神，不就是觉得我是那种人吗？"

主角张了张嘴，却发现自己发不出声音。

是啊，他刚才确实那么想了一秒。仅仅一秒。

"我......"

"你什么？"苏晴打断他，声音开始发抖，"你想说你没那么想？那你质问我干什么？"

主角松开了桌沿。掌心全是汗。

他想说对不起，想说他是太着急了，想说他是被那些流言气昏了头。但所有的话都堵在喉咙里，一个字都吐不出来。

"算了。"苏晴站起来，椅子在地板上划出刺耳的声音，"反正我在你心里，也就那样。"

"苏晴......"

"别跟着我。"

她转身就走，步伐很快，像是怕自己会后悔。主角看着她的背影消失在门口，忽然注意到她的肩膀在微微发抖。

[她哭了。]

这个念头让他觉得心脏被什么东西狠狠攥了一下。

他想去追，但脚像生了根。

窗外开始下雨了。
```

### 示例D：日常温馨场景
**适用场景**：种田文、日常文、治愈系、家庭伦理

骨架：一家人吃晚饭
```
主角一家人围坐吃晚饭 → [环境][对话][细节]
```

扩写要点：
- **[环境]** 生活气息细节（饭菜香、碗筷碰撞声、灯光）
- **[对话]** 自然家常，不需要每句话都有意义
- **[节奏]** 舒缓，适当留白
- **[细节]** 小动作、习惯性动作（父亲总是把肉夹给孩子）

扩写示范：
```
"开饭了——"

母亲的声音从厨房传来，伴随着碗筷碰撞的清脆声响。主角放下手机，起身去端菜。

餐桌上已经摆好了三副碗筷。父亲坐在老位置上，正用遥控器调着电视音量。那是他的习惯，新闻联播必须听得清清楚楚。

"今天做了什么好吃的？"主角问。

"糖醋排骨，还有你最爱喝的番茄蛋汤。"母亲端着盘子走出来，额头上有细密的汗珠，"去叫你爸洗手，就知道看电视。"

"知道了。"

父亲恋恋不舍地放下遥控器，去洗手了。主角趁机偷吃了一块排骨，烫得直吐舌头。

"馋猫。"母亲笑着拍了他一下，"等大家坐齐了再吃。"

一家三口终于坐定。

父亲照例先给母亲夹了一块排骨，然后再给主角夹。这是他们家的规矩，几十年没变过。

"今天工作怎么样？"父亲问。

"还行吧，就那样。"主角扒拉着碗里的饭。

"什么叫就那样？"父亲皱起眉，"具体说说。"

"哎呀，孩子刚下班，让他先吃饭。"母亲打圆场，"工作上的事吃完饭再说。"

父亲"嗯"了一声，不再追问。

电视里新闻联播的声音作为背景音，母亲偶尔会评论两句时事，父亲应和着。主角埋着头吃饭，听着这些再熟悉不过的对话，忽然觉得心里踏实得很。

汤还冒着热气。

窗外的天慢慢黑了，家里的灯暖黄暖黄的。

这就是家。
```

### 示例E：体育/竞技场景（原篮球示例保留）
**适用场景**：系统流、竞技文、体育文

骨架：主角投篮命中
```
主角起跳投篮 → [动作][围观][系统]
```

扩写要点：

```
王浩的嘴角还挂着那抹嘲讽的笑，但已经开始僵硬。

李雷从床上坐了起来。

唰——

空心入网。

篮球穿过篮网的声音在突然安静的宿舍里格外清晰。

"我操......"王浩的遥控器掉在了地上。

主角落地，面无表情地看了他一眼。

"还打吗？"
```

## 去AI化专项指南

### 避免高频连接词

| AI高频词 | 人工替换方案 | 使用场景 |
|---------|------------|---------|
| 此外/另外 | 再说、还有一点、话说回来 | 补充信息 |
| 然而/但是 | 但、不过呢、问题是、可实际上 | 转折 |
| 因此/所以 | 所以、结果就是、这样一来 | 因果 |
| 综上所述 | 说白了、老实说、说到底 | 总结 |
| 与此同时 | 这边...那边...、一边...一边... | 并行 |
| 非常重要 | 很关键、这事不能忽视、说白了 | 强调 |
| 值得注意的是 | 有意思的是、你发现没、巧了 | 引出重点 |
| 换句话说 | 说白了就是、意思就是说、等于说 | 解释 |

### 增加"人味"技巧

**1. 适度口语化**
- 使用语气词："哦"、"呢"、"啊"、"嘛"、"呗"
- 示例："这能行吗？"他嘀咕着，"我总觉得不太靠谱呢。"

**2. 轻微瑕疵（故意为之）**
- 轻微重复："他想了想，又想了想，还是决定..."
- 口语化啰嗦："这块可能说乱了，意思就是..."
- 短句不完整："明白了。就这么干。"

**3. 主观表达**
- 使用第一人称感受："我觉得"、"说实话"、"老实说"
- 加入个人判断："这招挺损的，但好用。"

**4. 具体细节植入**
- 用具体数据代替笼统描述："投篮命中率不到三成"代替"投篮很差"
- 加入时间/地点/人物细节："昨天下午三点，在体育馆二楼"

**5. 打破规整性**
- 避免连续使用相同长度的句子
- 偶尔使用倒装、插入语、省略句
- 让对话更自然，不必每句都完整
