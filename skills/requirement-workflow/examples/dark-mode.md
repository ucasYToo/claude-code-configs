# PRD: 暗黑模式切换

版本: v1.0.0 | 日期: 2026-04-06

---

## 概述

在用户设置中添加暗黑模式切换，具有系统偏好检测和即时主题切换。

---

## 研究发现

- **现有**: 代码库中没有当前主题系统
- **模式**: CSS 变量 + data 属性方法是标准
- **库**: Tailwind CSS 有内置的深色模式支持（我们已经在使用 Tailwind）
- **实施**: 在 Tailwind 配置中使用 `darkMode: 'class'`

---

## 需求

- [ ] `/settings/appearance` 页面中的切换
- [ ] 默认尊重系统 `prefers-color-scheme`
- [ ] 在 localStorage 中持久化用户选择
- [ ] 无需页面重新加载即可应用主题
- [ ] 主题之间平滑过渡（300ms）

---

## 验收标准

- [ ] 切换在亮色/暗色/系统选项之间切换
- [ ] 用户选择在会话之间持久化
- [ ] 页面加载时没有错误主题的闪烁
- [ ] 适用于 Chrome、Firefox、Safari（最近 2 个版本）
- [ ] 通过两种主题的 WCAG 对比度要求

---

## 约束

- **技术**: 必须使用 Tailwind CSS
- **兼容性**: 支持 CSS 变量的浏览器
